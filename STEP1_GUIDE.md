# STEP 1 Guide

This document provides a comprehensive guide for the institutional quant system, outlining the structure and expected functionalities for the Java execution engine and Python Flask ML API.

## Java Execution Engine
- **TradeExecutor.java**: Responsible for executing trades based on signals.
- **FeatureGenerator.java**: Generates features for trading.
- **Trade.java**: Represents a trade object.
- **Main.java**: Entry point of the execution engine.

## Python Flask ML API
- **app.py**: Flask application to serve the ML model.
- **ml_model.py**: Handles machine learning model loading and prediction.
- **requirements.txt**: Lists the dependencies for the API.
